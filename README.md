# Evolutionary Functional Liver Axis Discovery

## 项目目标

本项目面向**脊椎动物肝脏空间多组学数据**（空间转录组、空间代谢组、空间坐标及可选形态学特征），
在**不依赖血管标记或解剖模板**的条件下，推断肝脏灌流相关功能空间组织是否存在。

核心科学问题：

> PV–CV 循环结构在脊椎动物中普遍保守存在，但其是否被“功能化”为可检测空间分区（zonation-like organization）可能随进化变化。

---

## 生物学前提与建模边界

- 假设保守成立：
  - 门静脉样入流（PV-like inflow）
  - 中央静脉样出流（CV-like outflow）
  - 灌流路径 PV → sinusoid → CV
- 不假设保守成立：
  - 哺乳动物经典小叶几何模板
  - 单一全局 zonation 轴
  - 固定的三联体结构

因此，算法聚焦于：

- 是否存在与血流方向一致的功能势场组织；
- 这种组织是单轴、局部多轴、弱梯度还是缺失。

---

## 核心建模思想：连续功能势场

组织功能状态建模为连续势场：

- `u(x)`：定义在空间坐标 `x` 上的功能势能。

`u(x)`由多模态共同约束：

- RNA 表达状态
- 代谢状态
- 可选形态特征
- 空间平滑先验

若循环驱动功能分区存在，则 `u(x)` 应表现为从 PV-like 到 CV-like 的平滑变化。

---

## 结构假设：multi-axis 而非 single-axis

真实组织可由多个局部 PV–CV 功能单元重复构成：

- CV-like 区域 ≈ 局部高势点
- PV-like 区域 ≈ 局部低势点
- 局部功能轴由 `∇u(x)` 的梯度流定义，而不是单一主方向。

---

## 计算任务定义

### 1) 学习跨模态一致的势场 `u(x)`

要求：

- 空间连续与平滑
- RNA / 代谢（/形态）共同支持
- 不依赖解剖标记

### 2) 判定空间组织类型

输出四分类：

- `Global axis`
- `Multi-local axes`
- `Weak organization`
- `Absent`

### 3) 恢复局部功能结构

基于 `u(x)` 推断：

- 局部极值（PV-like / CV-like）
- 功能盆地（lobule-like 单元）
- 局部轴方向（`∇u`）

### 4) 验证跨模态一致性

同一结构需在 RNA 与代谢中共同支持，避免单模态噪声伪影。

---

## 跨物种适配要求

算法必须具备：

- 几何模板无关
- 结构对齐无关
- 可比较伪空间输出

标准化输出：

- `u*(x) ∈ [0, 1]`

用于跨物种比较：

- 全局轴强度
- 局部轴数量
- 跨模态一致性

---

## 参考算法流程（实现导向）

1. **数据预处理**
   - RNA、代谢各自归一化与降噪
   - 坐标标准化
2. **多模态表示学习**
   - 构建共享潜变量 `z(x)`（可用对比学习或图融合）
3. **势场估计**
   - 在空间图上学习标量场 `u(x)`
   - 目标函数包含：
     - 模态重建/对齐损失
     - 空间拉普拉斯平滑
     - 梯度结构稀疏或稳定约束
4. **组织模式判定**
   - 全局轴统计：主方向解释度、梯度一致性
   - 局部轴统计：极值点个数、盆地分割数
5. **跨模态一致性检验**
   - RNA-only 与 metabolite-only 势场相关性
   - 置换检验评估显著性
6. **跨物种比较**
   - 以 `u*(x)` 及派生指标建立进化比较矩阵
