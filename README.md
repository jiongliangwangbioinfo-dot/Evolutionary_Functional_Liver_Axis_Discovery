# Evolutionary Functional Liver Axis Discovery

## 项目目标

本项目面向**脊椎动物肝脏空间多组学数据**（空间转录组、空间代谢组、空间坐标及可选形态学特征），
在**不依赖血管标记或固定解剖模板**的条件下，推断肝脏灌流相关功能空间组织是否存在。

核心问题不是“有没有 PV/CV”，而是：

> PV–CV 循环结构是否被功能化为可检测的空间分区（zonation-like organization）。

---

## 生物学前提与建模边界

- 假设保守成立：
  - 门静脉样入流（PV-like inflow）
  - 中央静脉样出流（CV-like outflow）
  - 灌流路径 PV → sinusoid → CV
- 不假设保守成立：
  - 哺乳动物经典小叶几何模板
  - 单一全局 zonation 轴
  - 固定三联体结构

因此，算法应输出“功能组织是否存在及其结构类型”，而不是硬套形态模板。

---

## 总体建模思想

将组织建模为连续功能势场 `u(x)`：

- `u(x)` 由 RNA + 代谢（+形态）共同约束；
- 若存在循环驱动功能分区，则 `u(x)` 在空间上应呈现平滑方向性变化；
- 真实组织可能是多局部轴重复（multi-axis），而非单轴。

---

## 你提出的 5 步分解：可行，但建议做“无标记鲁棒化”改造

你的分解：

1. 识别空间组织的血管位置
2. 建立每个血管与其邻近血管的轴
3. 计算每个轴下不同基因和代谢物表达
4. 对轴聚类并调整方向
5. 建立 PV-CV 轴

**结论：可行。**
但需要处理一个关键限制：本项目不依赖显式血管标记，因此第 1 步应解释为**识别“血管样功能锚点（vascular-like anchors）”**，而非病理学真血管分割。

### Step 1. 识别 vascular-like anchors（替代“直接找血管”）

可结合三类证据：

- 空间拓扑：局部流形中心性、边界结构、稀疏通道特征；
- 多组学异常：代谢交换相关模块和转录梯度拐点；
- 形态学可选证据：管腔样纹理（若有图像）。

输出：候选锚点集合 `A = {a_k}`，并保留不确定度分数。

### Step 2. 构建局部 anchor-pair 轴

- 在空间图上连接邻近锚点对 `(a_i, a_j)`；
- 使用最短路/扩散测地距离定义轴路径；
- 对每条轴建立局部伪空间坐标 `s_ij ∈ [0,1]`。

输出：局部轴集合 `Ξ = {ξ_t}`。

### Step 3. 轴内表达剖面估计

对每条 `ξ_t`：

- 计算基因、代谢物沿 `s_t` 的平滑表达曲线；
- 评估单调性、非线性形状、峰谷位置；
- 计算 RNA/代谢跨模态一致性（相关、互信息、共变曲线相似度）。

输出：每条轴的“功能签名向量” `f_t`。

### Step 4. 轴聚类与方向统一

- 先按 `f_t` 聚类得到轴类型（强梯度、双峰、弱梯度等）；
- 再做方向对齐：
  - 以跨模态一致性最大化为目标；
  - 或以全局势场 `u(x)` 梯度方向作为参考极性；
- 对方向不确定轴保留双向置信区间，不强行翻转。

输出：方向统一后的轴簇 `Ĉ`。

### Step 5. 从局部轴到 PV-CV 功能轴重建

- 将高置信局部轴投票到全局势场 `u(x)`；
- 将锚点按势场极值与连接关系分为 PV-like / CV-like 端；
- 形成“全局单轴”或“多局部轴网络”两种可能输出。

最终给出：

- `Global axis / Multi-local axes / Weak / Absent` 四分类；
- 标准化伪空间 `u*(x) ∈ [0,1]`；
- 跨模态一致性与显著性检验结果。

---

## 为什么这套分解在跨物种上更稳健

- 不要求固定几何模板；
- 允许局部重复单元和随机排布；
- 通过“局部轴 → 轴簇 → 势场”分层整合，避免一步法过拟合哺乳动物结构；
- 可比较输出统一为 `u*(x)` 与结构统计量。

---

## 最小可执行实现（建议）

1. 构建空间 kNN 图并学习多模态嵌入 `z(x)`；
2. 从 `z(x)` + 空间拓扑中找候选 anchor；
3. 枚举局部 anchor-pair 并计算轴内剖面；
4. 聚类 + 方向统一；
5. 回归全局势场 `u(x)` 并进行四分类判定；
6. 通过置换检验验证非随机性。

---

## `src/` 最小原型（模拟数据）

已提供可运行脚手架：`src/prototype.py`，实现流程：

1. anchor 检测（无标记）
2. 局部轴构建（anchor-pair 图测地路径）
3. 方向统一（基于跨模态一致性 + 代理势场）
4. 全局 `u(x)` 回归（轴端点约束 + 图平滑迭代）

运行示例：

```bash
python3 src/prototype.py --n-side 24 --seed 1 --k 8 --n-anchors 20
```
